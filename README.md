# Explorando Padrões de Projeto na Prática com Java

Este repositório contém exemplos práticos de **padrões de projeto (Design Patterns)** implementados em **Java**, desenvolvidos durante o laboratório **"Explorando Padrões de Projetos na Prática com Java"**.

O objetivo do projeto é demonstrar, de forma simples e didática, como alguns padrões de projeto podem ser aplicados para melhorar a **organização, reutilização e manutenção do código**.

---

## 📚 Padrões de Projeto Implementados

Os seguintes padrões foram explorados neste projeto utilizando **Java puro**:

### 🔹 Singleton

Garante que uma classe tenha **apenas uma única instância**, fornecendo um ponto global de acesso a ela.

### 🔹 Strategy

Permite definir **diferentes algoritmos ou comportamentos**, encapsulando-os em classes separadas e tornando-os intercambiáveis em tempo de execução.

### 🔹 Facade

Fornece uma **interface simplificada** para um conjunto de classes ou subsistemas mais complexos, facilitando sua utilização.
